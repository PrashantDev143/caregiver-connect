# Caregiver Connect

Caregiver and patient safety dashboard built with React, TypeScript, and Supabase.

## Getting started

```sh
# Step 1: Clone the repository.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd caregiver-connect

# Step 3: Install dependencies.
npm install

# Step 4: Start the development server.
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase
